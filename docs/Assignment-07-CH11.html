Exercise 4 Predator–prey

Questions

  1. Where there are many prey and few predators (high \(R\), low \(F\)), which way does the flow mainly point?
leh-highR-lowF

Mainly to the right (R increasing, F not changing much).

Mainly upward (F increasing, R not changing much).

Roughly northeast (both R and F increasing).

Mainly downward (F decreasing).

  1. Where there are few prey and many predators (low \(R\), high \(F\)), which way does the flow mainly point?
leh-lowR-highF

Mainly to the right (R increasing).

Roughly southwest (both R and F decreasing).

Roughly northeast (both R and F increasing).

Mainly upward (F increasing).

Mainly to the left (R decreasing only).

  1. How does long-term behavior here differ from SIR?
leh-vs-SIR

SIR has no state space; predator-prey does.

SIR trends one way; here the state can cycle and not settle.

SIR has two state variables; predator-prey has only one.

Predator-prey has no equilibrium; SIR always has one.

  1. When would the cycle break (populations no longer follow a repeating cycle)?
leh-cycle-break

If we start with too many prey.

One population goes to zero (extinction); the closed orbit is lost.

If we start exactly at the equilibrium.

If the cycle runs for more than one full loop.
