Exercise 4 AM and FM radio: multiplication vs. composition

You’ve probably heard of AM and FM radio. Both broadcast a signal (music, voice) by combining it with a high-frequency carrier wave. The math behind each method is a different way of combining two functions: AM uses multiplication; FM uses composition. We use the storybook function osc() (oscillator): osc(\(u\)) \(= \sin(2\pi u)\), so one cycle per unit of \(u\). Below we walk through both with R, using time \(t\) as the input.

Step 1: The carrier wave

The carrier is a high-frequency wave. We use osc(\(f_c\cdot t\)) with \(f_c = 5\), so \(C(t) =\) osc(\(5t\)): 5 cycles per unit time.

Step 2: The signal

The signal (e.g. audio) is a lower-frequency wave. We use \(S(t) =\) osc(\(t\)): 1 cycle per unit time, a stand-in for the message.

Step 3: AM — amplitude modulation (multiplication)

In AM, the amplitude of the carrier is varied by the signal. So the transmitted wave is

AM(\(t\)) = \(C(t) \cdot \bigl(1 + m\cdot S(t)\bigr)\).

Here \(m\) is the modulation depth (we use \(m = 0.7\)). So we multiply the carrier by a factor that depends on the signal: one function \(\times\) another, same input \(t\).

You can see the carrier’s oscillations, with their height (amplitude) rising and falling with the signal.

Step 4: FM — frequency modulation (composition)

In FM, the signal changes the argument (phase) of the carrier instead of its amplitude. So the transmitted wave is

FM(\(t\)) = osc\(\bigl(f_c\cdot t + k\cdot S(t)/(2\pi)\bigr)\).

So we compose: the outer function is osc, and its input is \(f_c\cdot t + k\cdot S(t)/(2\pi)\) — a function of \(t\) that includes the signal. We use \(k = 2\) so the effect is visible.

Here the frequency of the wave (how tightly the oscillations are packed) changes with the signal, while the amplitude stays constant.

Exercise 5 Combining storybook functions to make new shapes

We can build more complex shapes by combining two storybook functions in different ways: multiplication, linear combination (adding scaled versions), or composition (one function’s output as the other’s input). This exercise uses hill(\(x\)) (a symmetric bump) and hillside(\(x\)) (an S-curve that rises and levels off) as building blocks. All plots use the domain \(x \in [-2, 4]\) so both shapes are visible.

Step 1: The two building blocks

Step 2: Three ways to combine them

A. Multiplication: \(h(x) =\) hill(\(x\)) \(\cdot\) hillside(\(x\)). The output is large only where both are moderately large, so you get a single asymmetric bump that falls off on the left and right.

B. Linear combination: \(h(x) = 0.6\cdot\) hill(\(x\)) \(+ 0.4\cdot\) hillside(\(x\)). The shape is a blend of the two: a peak with a rising tail.

C. Composition (input shift): \(h(x) =\) hill(\(x - 2\)). The same hill shape, but the peak is shifted to the right (the argument to hill is \(x - 2\) instead of \(x\)).
