Exercise 3 Polynomials and monomials

A polynomial is a linear combination of a few simple building blocks. Those building blocks are called monomials. Using storybook names, the simplest monomials are: flat() (constant 1), steady(\(x\)) (just \(x\)), steady(\(x\))·steady(\(x\)) (i.e. \(x^2\)), and so on. So when we write a quadratic like \(ax^2 + bx + c\), we are choosing a linear combination of exactly three ingredient functions: the constant (flat), the identity (steady), and the square (steady·steady). The “poly” in polynomial means “many” of these monomial terms.

A standard example is height of a ball (in meters) as a function of time (in seconds): under constant gravity, height is a quadratic in \(t\), so we write it in storybook form as \(a\cdot\)steady(\(t\))·steady(\(t\)) \(+ b\cdot\)steady(\(t\)) \(+ c\cdot\)flat() — i.e. \(at^2 + bt + c\). Run the code below to see an example shape (\(2t^2 - 7t + 3\)). You can change the coefficients to see how the curve changes.

Exercise 4 AM and FM radio: multiplication vs. composition

You’ve probably heard of AM and FM radio. Both send a signal (e.g. music, voice) by combining it with a carrier wave. The two methods correspond to two ways of combining functions: AM uses multiplication; FM uses composition. We model both using the storybook function osc(): a fast osc as the carrier and a slower osc as the signal. Below we walk through both with R.

Step 1: The carrier

The carrier is a rapid oscillation. We use osc with input \(5t\) (five cycles per unit time). In the code, that’s osc(5·\(x\)).

Step 2: The signal

The signal is the “message” we want to send — here, a slower oscillation: osc(\(t\)), one cycle per unit time.

Step 3: AM — amplitude modulation (multiplication)

In AM, the height (amplitude) of the carrier is varied by the signal. We multiply the carrier by a factor that depends on the signal: same input \(t\) into both, then multiply their outputs. So the transmitted wave is carrier \(\times\) (1 + modulation·signal). We use a modulation factor of 0.7.

The carrier’s height rises and falls with the signal.

Step 4: FM — frequency modulation (composition)

In FM, the signal changes the argument of the carrier instead of its amplitude. So we compose: the outer function is osc (the carrier shape), and its input is something that depends on both time and the signal. The wave keeps roughly constant height, but how tightly the oscillations are packed varies with the signal.
